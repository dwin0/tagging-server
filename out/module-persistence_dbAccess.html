(inner) queryMultiple(statement, variables, callback)

queries the database with multiple variations of a single statement in parallel.
Parameters:
Name Type Description
statement string statement a placeholder like $1
variables Array array containing a value for every query like [ VAR_1, VAR_2, VAR3 ] which returns in 3 queries. Use makePoints or makeMultipoints to fit the required form.
callback function function which will be called with the result of type (error, array)
